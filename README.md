# 🚀 Nasdaq Lab: Real-Time Financial Monitoring

A high-performance Python environment for monitoring the **Nasdaq Composite (^IXIC)** and major tech equities. This lab combines deep historical fundamental data with live, low-latency market tracking for 2026.

## 🛠️ Tech Stack
* **Python 3.10+**
* **yfinance** (Live Market Streaming)
* **Nasdaq Data Link** (Institutional Fundamentals)
* **Linux DBus** (Native OS Notifications)

## 📊 Features
* **Real-Time Dashboard:** 10s refresh cycles for the Nasdaq and watchlist.
* **Terminal UI:** Color-coded price action (Green/Red) for instant visual feedback.
* **Hybrid Data Architecture:** Accesses both 2018 historicals and 2026 live pricing.
* **Secure Credentialing:** Uses Linux environment variables to prevent API key leaks.

## 🚀 Usage
```bash
python3 nasdaq_lab_v4.py
```
