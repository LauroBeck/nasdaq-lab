# Market-Pulse-2026 📈

High-performance C++ Intelligence Suite designed for **International Financial Management** simulations.

### 🚀 Features:
* **Capital Budgeting**: 5-Year NPV calculation with automated "Accept/Reject" logic.
* **Risk Management**: Real-time Stop-Loss triggers for Nasdaq/Bloomberg-style trading.
* **FX Exposure**: Forward contract hedging for EUR/USD cross-border capital aports.
* **Client Migration**: Batch processing simulation for 10k accounts (WFC -> JPM).

### 🛠 Tech Stack:
* **Language**: C++17
* **Build System**: Makefile
* **Logic Framework**: Jeff Madura Principles

### 🖥 Quick Start:
\`\`\`bash
make run
